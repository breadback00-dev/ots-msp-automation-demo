# Ship Prompt

Use this when implementation is complete and you want to prepare for merge/release.

```text
Prepare this work for shipping.

Do the following:
- Run checklists/pre-ship-checklist.md.
- Review the diff.
- Run relevant tests/build/lint/typecheck.
- Check docs or examples that need updating.
- Create a concise change summary.
- Note risks, migrations, rollout concerns, and follow-up items.
- Draft a PR description using templates/project/pr-description.md.

Scope:
<describe branch, feature, or changes here>
```
