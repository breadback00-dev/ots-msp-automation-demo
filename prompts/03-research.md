# Research Prompt

Use this before choosing libraries, APIs, models, plugins, GitHub projects, deployment platforms, or security-sensitive approaches.

```text
Research this before recommending a path.

Follow the research source priority in AGENTS.md (official docs first, then primary GitHub repos, then changelogs/issues, secondary articles last).

For each serious option, include:
- what it is best for
- maintenance/activity signal
- integration cost
- risks and limitations
- whether it is overkill here
- recommendation and why

Also say whether any plugin, skill, GitHub repo, MCP/app connector, or local script would materially improve the work.

Save durable findings as a research note (templates/project/research-note.md) in docs/research/.

Research topic:
<describe topic here>
```
