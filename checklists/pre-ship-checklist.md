# Pre-Ship Checklist

- Goal still matches the original request.
- Happy path works.
- Important edge cases are handled.
- Error/loading/empty states are handled where relevant.
- Auth/permissions/data access are correct.
- Tests cover changed behavior or the gap is explained.
- Build/typecheck/lint status is known.
- Diff contains no unrelated refactors.
- Docs or examples are updated if needed.
- Rollback/migration risk is understood.
