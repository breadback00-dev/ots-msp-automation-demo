# Implementation Checklist

- Read `AGENTS.md`.
- Read relevant docs under `docs/`.
- Search for similar code.
- Identify test command(s).
- Make scoped changes.
- Add or update tests when behavior changes.
- Run focused tests first.
- Run broader checks if the change is risky.
- Review the diff.
- Update docs/templates if behavior or workflow changed.
- Summarize changes and verification.
