# Research Checklist

Use current research when the decision depends on changing external reality.

## Research Needed

- Library/package choice.
- Framework/API behavior.
- AI model or provider choice.
- Security or compliance practice.
- Deployment or hosting behavior.
- Pricing, quotas, rate limits, or licensing.
- Browser/mobile/platform support.
- GitHub examples or maintained templates.

## Source Priority

Follow the source priority defined in `AGENTS.md` (official docs first, primary GitHub repos second, changelogs/issues third, secondary articles last).

## Capture

Save durable findings in `docs/research/` using `templates/project/research-note.md`, with links, dates, and recommendation.
